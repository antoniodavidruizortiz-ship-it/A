🚀 Cómo subir a GitHub Pages
1
Ve a github.com → New repository → ponle nombre tipo caskia → Public → Create
2
Dentro del repo: Add file → Upload files → sube este caskia-pwa.html
3
Renombra el archivo a index.html (o súbelo ya con ese nombre)
4
Settings → Pages → Source: Deploy from branch → Branch: main → / (root) → Save
5
En ~2 minutos tendrás tu URL: tuusuario.github.io/caskia
6
Esa URL la metes en pwabuilder.com para generar el APK de Android
🔑 Configuración rápida
Edita las 3 primeras líneas del <script> en el HTML:
const MISTRAL_KEY = 'tu-key-aqui';
const MISTRAL_MODEL = 'mistral-small-latest';
const PC_WORKER = 'https://tu-worker.workers.dev';
Consigue tu key en console.mistral.ai
⚠️ Seguridad
❗ Nunca compartas ni pegues tu API key en chats o foros.
❗ Si la expones accidentalmente, bórrala en console.mistral.ai y crea una nueva.
❗ Para producción, usa un proxy backend en lugar de la key directa en el HTML.
